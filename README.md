# nfc_java_library


  这是一个用于  NFC 芯片（13.56Mhz， 14443TypeA ）的  Android SDK. 用来方便Android App 对 NFC 芯片进行读、写、锁定、认证、更换秘钥等操作.
  目前该 SDK 支持的芯片类型包括  FJ8018, NXP203, NXP216,M1, UltraLight 等.
